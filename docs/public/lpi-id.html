LPI IDの取得手順
LPIC 受験に必要な LPI ID を取得するための手順です
LPIC を受験するには、事前に LPI ID（受験者を識別するID）を取得しておく必要があります。
以下の3ステップに沿って公式サイトでアカウント登録を行い、ご自身の LPI ID を確認してください。
STEP 1 アカウント登録ページにアクセス

以下のリンクから公式のアカウント登録ページにアクセスしてください。

公式アカウント登録ページ（cs.lpi.org）を開く ↗
STEP 2 登録情報の入力

画面下部に表示される項目を入力してください。

入力時の注意点
  • 所属の会社/学校 は空欄で構いません
  • 引越しが控えている方は現住所を入力してください（引越し後はマイページから変更可能です）
  • ユーザー名 は一意の値である必要があるため、名前_eightbit_co のように他者と重複しない値を設定してください
  • ご自身で設定したパスワードは、必ず忘れないようメモしておいてください

LPIからのメール配信を希望しますか？ はチェックせず、プライバシーポリシー は「はい」を選択して「登録」ボタンをクリックすると、アカウント作成は完了です。

STEP 3 LPI ID の確認

以下のリンクからログインページにアクセスしてください。

公式ログインページ（cs.lpi.org）を開く ↗
入力時の注意点
  • ユーザー名 欄には、先ほど登録したメールアドレスを入力してください

ログインに成功するとマイページが表示されます。画面左上に表示される LPI から始まる文字列が LPI ID ですので、控えておいてください。

以上で LPI ID の取得は完了です。
受験申込の際に必要となりますので、大切に保管してください。
